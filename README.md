Notification 4
Description
This project implements a flexible notification system using Java.
Files

Main.java - Entry point
Alertsystem.java - Alert management
NotificationMedium.java - Notification interface/base class
EmailService.java - Email notification implementation
SMSService.java - SMS notification implementation

Requirements

Java 17 or later

How to Run

Download or clone the repository.
Open the project in IntelliJ IDEA or another Java IDE.
Run Main.java.
